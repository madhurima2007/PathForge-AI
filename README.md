
# 🚀 PathForge AI – Career Roadmap Tracker

A modern and interactive Career Roadmap Tracker built with **HTML, CSS, and JavaScript** that helps students, aspiring developers, and tech enthusiasts explore Computer Science career paths through visually appealing roadmaps.

## 🌟 Overview

PathForge AI is designed to simplify career planning in technology by providing structured roadmaps for multiple Computer Science domains. Users can browse different career paths, explore the required skills, and understand the progression needed to achieve their goals.

The application features a modern glassmorphism-based interface, responsive design, domain search functionality, and interactive roadmap visualization.

## 🚀 Getting Started

### Option 1: Open Directly

Simply open:

```text

```

in your browser.

### Option 2: Run on Localhost

Using VS Code:

1. Install Live Server extension
2. Right-click `index.html`
3. Click **Open with Live Server**

The project will launch automatically.
